# Hidera ClearView

Allows you to hide icons, the Windows taskbar, and other elements from the desktop for a clean look.

> **The full source code for this project is currently being prepared and will be made publicly available here very soon!** Stay tuned for updates.
---

## About the Project

Hidera ClearView is a simple, lightweight utility designed for those who have a lot of shortcuts and folders on their desktop but still want everything to look aesthetic and clean. 

The program runs silently in the background and is currently **still in active development**. Community support, testing, and feedback are highly appreciated!

## Configuration Settings

<p align="center">
  <img src="https://github.com/user-attachments/assets/597a2d6b-b9a4-4024-9c95-f33785d9db17" width="320" alt="Hidera Settings Panel" />
</p>

You can completely customize your experience using the native built-in settings panel:

* **Component Toggles:** Choose exactly what you want to disappear — the Windows Taskbar, standard Desktop Icons, or active Rainmeter widgets.
* **Hotkey Customization:** Adjust the hotkey trigger conditions directly. You can toggle **Win** and **Alt** modifier checkboxes independently and pair them with any functional key from **F1** to **F12**.
* **Smart Desktop Focus:** The global hotkey trigger safely validates active window classes. It fires **only** when you are interacting directly with the empty desktop space, staying completely non-intrusive during full-screen games, media players, or web browsers.
* **Proactive RAM Efficiency:** After executing a visibility toggle, the background process actively flushes its memory working set down to a minimal profile of **~16 - 22 MB**.
